The aimp_sacd is a plugin that produces an ability to decode hi-res audio in the DSDIFF / DSF formats via AIMP3.60 or newer.
The plugin was based on source code of the SACD Decoder plugin that licensed under LGPLv2.

The libsacd is compilled version of two libraries - libdsd2pcm and libdstdec.
